# Cozy Cover Beddings

This is a premium e-commerce bedding website built with React, Vite, and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features

- **Home**: Hero section, Best sellers, Newsletter.
- **Shop**: Filtering, Sorting, Product Grid.
- **Product Detail**: Gallery, Variants, Related Items.
- **Cart**: Management, Totals.
- **Checkout**: Form validation, Success state.
